# knowledge-work-plugins

A collection of Claude skills for knowledge work, installable across Claude Code, Claude Cowork, and claude.ai.

## Plugins

### rich-life

Your guide to building a Rich Life, powered by Ramit Sethi's philosophy from "I Will Teach You To Be Rich" and "Money for Couples". Covers budgeting, debt elimination, investing, and building a Conscious Spending Plan.

## Installation

### Claude Code (CLI)

```bash
claude plugin marketplace add CarlosEspejo/knowledge-work-plugins
claude plugin install rich-life@carlos-skills
```

Then just chat — the skill auto-triggers on financial topics, or invoke it explicitly with `/rich-life`.

### Claude Cowork (Desktop App)

1. Open the Claude desktop app → Cowork
2. Go to Settings → Plugins
3. Add marketplace: `CarlosEspejo/knowledge-work-plugins`
4. Install the `rich-life` plugin

Then chat naturally or use `/rich-life` to start a session.

### Claude.ai / Claude Desktop Chat

1. Go to claude.ai → click your profile → **Customize** → **Skills**
2. Click **Add skill** → **Upload skill**
3. Upload the [rich-life SKILL.md](plugins/rich-life/skills/rich-life/SKILL.md) file directly

> **Note:** The skill instructions won't auto-update when this repo changes. Check back here for the latest version.
